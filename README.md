# 🎵 Spotify-Inspired Music Player

A simple Spotify-inspired music player built using **HTML, CSS, and Vanilla JavaScript**.  
This project focuses on core audio playback functionality with a clean and responsive UI.

---

## Features

-  Play / Pause music control
-  Next and Previous track navigation
-  Clickable playlist for instant song selection
-  Fully responsive and modern UI design
-  Spotify-inspired interface layout
- Runs locally in the browser (no backend required)

##  Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Audio API (HTML5 Audio Element)

## Learning Outcomes

- Learned DOM manipulation using JavaScript  
- Understood HTML5 Audio API  
- Improved UI/UX design skills  
- Practiced event handling and state management in frontend
